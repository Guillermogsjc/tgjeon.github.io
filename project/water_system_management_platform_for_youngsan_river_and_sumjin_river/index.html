Descriptions

Description and contributions

In this project, I have contributed to developed integrated system for watershed manager. In developed systems, the calibrated watershed model was applied to execute scenario analysis and various environmental information about Youngsan and Seomjin River was also implemented

  • Development of core systems for integrating operation
    • Management of user, document, file, multimedia, and network in integrating operation system
    • Design and development to set up a system environment for integration
  • Development of web-based core systems for integrating operation
    • Development of graphical user interface (GUI)
    • Use and development of interpreter/builder of web-integrating system
    • Design and development to set up a web-based system environment for integration
  • Design of system for data process from real-time data of water quality and volumetric flow rate
    • Analysis from a questionnaire of user
    • Investigation of data structure and quality credibility from real-time (raw) data
    • Investigation of data structure from observed data
    • Comparison and verification of data between real-time data and observed data
    • Construction of reliable raw data from systematic data analysis
    • Selection and development of models for data process
    • Investigation of credibility of secondary data (information)
    • Analysis and modification of errors in the models
    • Design of a system for data process
  • Development and integration of systems
